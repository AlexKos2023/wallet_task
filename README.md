# Программа для контроля доходов и расходов

Простая консольная программа на Python для учёта личных доходов и расходов.  
Учёт ведётся только в целых рублях, баланс может быть отрицательным (можно использовать кредитные средства).

---

## Как использовать

1. Установить Python на ПК.
2. Скопировать файл программы на компьютер.
3. Открыть командную строку.
4. Перейти в директорию, где находится файл программы.
5. Запустить программу:

```bash
python finance_tracker.py
```

---

## Функционал

При запуске программа входит в бесконечный цикл и показывает главное меню с возможными действиями.  
Пользователь выбирает действие, вводя соответствующий номер.

- При первом запуске запрашивается начальный остаток денег.
- Доступные действия:
  - отображение текущего баланса, расходов и доходов;
  - добавление новой записи (доход или расход);
  - изменение рабочего каталога и создание/обновление файла данных (при необходимости снова вводится остаток денег);
  - поиск информации;
  - изменение существующих записей;
  - выход из программы.
