# Финансовый трекер: скрипт для учёта доходов и расходов

Простой консольный скрипт на Python для ведения личного финансового дневника:  
отслеживания баланса, добавления записей, поиска, редактирования и расчёта общей суммы доходов и расходов.

---

## Функционал

- Управление рабочей директорией и автоматическое создание каталога `Записки`.
- Создание и хранение записей в файле `data.txt` в формате JSON.
- Учёт текущего баланса.
- Добавление записей с полями:
  - дата (`ГГГГ-ММ-ДД`),
  - категория (`доход`/`расход`),
  - сумма (целое число > 0),
  - описание.
- Поиск записей по полю: дата, категория или сумма.
- Редактирование существующей записи с автоматическим пересчётом баланса.
- Расчёт и вывод:
  - текущего баланса,
  - общей суммы доходов,
  - общей суммы расходов.

---

## Структура кода

Скрипт состоит из следующих функций:

### 1. Работа с директорией

- `get_path()` — позволяет выбрать текущую или новую рабочую директорию.
- `create_directory_and_file()` — создаёт папку `Записки` и файл `data.txt`,  
  записывает туда начальный баланс (остаток денег).

### 2. Управление балансом

- `get_current_balance()` — читает первую строку файла `data.txt`, парсит JSON и возвращает текущий баланс.
- `update_balance(category: str, amount: int, current_balance: int)` — обновляет баланс в зависимости от типа операции (`доход` или `расход`).

### 3. Валидация данных

- `is_valid_date(date_str: str)` — проверяет, что дата в формате `ГГГГ-ММ-ДД` и год ≥ 2024.
- `is_valid_category(category: str)` — проверяет, что категория — `доход` или `расход`.
- `is_valid_amount(amount: str)` — проверяет, что сумма — положительное целое число.

### 4. Работа с записями

- `add_entry()` — запрашивает у пользователя дату, категорию, сумму и описание,  
  добавляет новую запись в файл и обновляет баланс.
- `search_entries()` — ищет записи по полю (`дата`, `категория`, `сумма`) и выводит результаты с ID.
- `update_entry()` — позволяет изменить существующую запись по её ID,  
  автоматически пересчитывает баланс.
- `calculate_finances()` — читает все записи, суммирует доходы и расходы, выводит баланс и сводку.

### 5. Главное меню

- `main_menu()` — консольное меню с выбором действия:
  - 1 — Баланс,
  - 2 — Добавить запись,
  - 3 — Изменить рабочий каталог,
  - 4 — Найти сведения,
  - 5 — Изменить сведения,
  - 6 — Выход.

## Как запустить

1. Убедитесь, что у вас установлен Python (версия ≥ 3.6).
2. Сохраните код в файл, например `finance_tracker.py`.
3. Запустите в терминале:

   ```bash
   python finance_tracker.py
   ```

4. При первом запуске будет:
   - предложено выбрать рабочую папку,
   - создана папка `Записки` и файл `data.txt`,
   - запрошена начальная сумма остатка денег.

---

## Технические особенности

- Данные хранятся в обычном текстовом файле `data.txt`, каждая запись — отдельная строка в формате JSON.
- Первая строка файла содержит словарь с текущим балансом:
  ```json
  {"Остаток денег": 10000}
  ```
- Остальные строки — записи вида:
  ```json
  {"дата": "2024-05-01", "категория": "доход", "сумма": 5000, "описание": "Зарплата"}
  ```
- Все операции (доход/расход) корректно пересчитывают баланс при добавлении и редактировании записей.
